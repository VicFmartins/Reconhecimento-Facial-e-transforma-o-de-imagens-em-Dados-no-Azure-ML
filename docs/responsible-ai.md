# Responsible AI

Este projeto foi reposicionado para um uso seguro de visao computacional.

## O que ele faz

- detecta possiveis rostos em uma imagem
- extrai metadados visuais simples
- anonimiza rostos com blur

## O que ele nao faz

- identificacao biometrica
- verificacao de identidade
- reconhecimento de pessoas especificas
- atribuicao de emocao, genero ou qualquer inferencia sensivel

## Motivo

Esse recorte reduz risco de uso indevido e deixa o projeto mais alinhado com privacidade, moderacao e boas praticas de IA responsavel.
